# agrinho-2026

A página é um site institucional educativo voltado para o Projeto Agrinho 2026, que traz o tema Agro Forte, Futuro Sustentável. O site possui um design moderno e interativo estruturado em seções Sobre, Tecnologia e Sustentabilidade que buscam conscientizar o visitante sobre a importância do agronegócio moderno, da tecnologia aplicada ao campo e da conservação ambiental. Ele conta ainda com elementos dinâmicos, como um botão para exibir mensagens e um contador simbólico de árvores preservadas.
